## 💻 Sobre o projeto
O Dramaflix foi criado no desafio do bootcamp da Digital Innovation One, onde a proposta era recriar a interface da Netflix dando uma nova roupagem ao site.
O Dramaflix foca nas séries de TV asiáticas.
